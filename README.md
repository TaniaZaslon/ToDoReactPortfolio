### About the project: 
Classical TODO list with an ability to add and remove tasks. 
Ability to color-code tasks.
A card for each task with a checklist.
And the ability to mark completed tasks.

### Stack:

- ReactJS (useState, useReducer, useEffect)
- React Router
- Axios
- Classnames
